# CXXQueue

A lock-free SPSC queue.

This class is thread safe when used with a single producer and a single consumer.

> [!NOTE]
> C++20 is required.

## Installation

### Swift Package Manager

Add a package dependency to https://github.com/sbooth/CXXQueue in Xcode.

### Manual or Custom Build

1. Clone the [CXXQueue](https://github.com/sbooth/CXXQueue) repository.
2. `swift build`.
